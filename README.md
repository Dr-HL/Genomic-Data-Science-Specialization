##Genomic Data Science Specialization
```
This specialization will teach you,
-- To understand, analyze, and interpret data from next-generation sequencing experiments. 
-- You will learn common tools of genomic data science, including Python, R, Bioconductor, and Galaxy. 
-- These courses can serve as,
(i) a stand-alone introduction to genomic data science 
[or]
(ii) It can compliment to a primary degree or postdoc in biology, molecular biology, or genetics.
```
```
The courses under this specialization:-

[1] Introduction to Genomic Technologies

[2] Genomic Data Science with Galaxy

[3] Command Line Tools for Genomic Data Science

[4] Python for Genomic Data Science

[5] Algorithms for DNA Sequencing

[6] Bioconductor for Genomic Data Science

[7] Statistics for Genomic Data Science

[8] Genomic Data Science Capstone (not enrolled in signature track)
```
